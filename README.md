# Data-Driven Investment Thesis: Goldman Sachs

## Project Overview
This project develops a data-driven investment thesis on Goldman Sachs by analyzing its business model, financial performance, valuation, and risk profile using publicly available data and Python-based analysis.

The objective is to simulate real-world analyst thinking and support evidence-based investment decision-making under uncertainty.

## Key Questions
- How strong is Goldman Sachs’ financial performance across market cycles?
- Does its valuation reflect its risk and earnings profile?
- What are the key risks that affect long-term returns?

## Tools & Technologies
- Python (pandas, numpy, matplotlib)
- Jupyter Notebook
- Public financial statements
- Market data (Yahoo Finance)

## Project Structure
- `data/` – raw and cleaned datasets
- `notebooks/` – analysis notebooks
- `visuals/` – generated charts and plots
- `thesis/` – written investment thesis and executive summary
